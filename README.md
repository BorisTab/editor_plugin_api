# Editor Plugin API

## Структура поставки плагина

```
pending
|_MyPlugin
| |_include
| | |_MyPlugin
| |   |_MyPlugin.h
| |_src
| | |_MyPlugin.cpp
| |_icon.xxx
| |_PLUGIN_INFO
| |_README.md
```

```
plugins
|_MyPlugin
| |_MyPlugin.so
| |_icon.xxx
| |_PLUGIN_INFO
| |_README.md
```
